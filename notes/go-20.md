# go

Learned something useful about go today.

## Key takeaway

This interacts with git in a non-obvious way.

## Details

I spent a while debugging this. The root cause was that go caches aggressively by default, and when your git configuration changes, the stale cache causes confusing behavior. Adding a cache-busting parameter fixed it.

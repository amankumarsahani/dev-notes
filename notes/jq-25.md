# jq

Learned something useful about jq today.

## Key takeaway

The default behavior is not what I expected - need to be explicit about configuration.

## Details

The trick is to separate the read and write paths. jq handles reads well out of the box, but writes need explicit transaction management. Without it, you get partial updates under concurrent load.
